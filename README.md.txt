# Node.js CI/CD Pipeline using GitHub Actions & Docker

This project automates code deployment using a CI/CD pipeline.

## Tools Used
- GitHub Actions
- Node.js
- Docker & DockerHub

## Workflow Steps
1. Trigger on push to main
2. Install dependencies
3. Build Docker image
4. Push image to DockerHub

## Repository Secrets
- `DOCKERHUB_USERNAME`
- `DOCKERHUB_TOKEN`
